선택자

CSS를 적용하고자 하는 요소를 선택하기 위한 문법

전체 요소 선택자

html의 모든 요소 선택
big 클래스 사용

타입 선택자

특정 태그를 사용한 요소 선택
bold 클래스 사용

클래스 선택자

특정 클래스명을 가진 요소 선택
class 속성(html)을 통해 클래스명 지정 가능
class 이름 앞에 .을 붙여 선택자 사용
제목에 blue 클래스로 폰트컬러 적용
이 문단에 big bold 클래스 둘 다 적용

아이디 선택자

특정 아이디를 가진 요소 선택
html 요소의 id 속성을 통해 아이디 지정.
id는 유일한 요소이다.
id 이름 앞에 # 붙여 선택자 사용
제목에 small id 사용, 폰트 5px
